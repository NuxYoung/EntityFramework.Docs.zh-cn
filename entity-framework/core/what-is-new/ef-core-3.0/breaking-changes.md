### <a name="dbfunctionschema-being-null-or-empty-string-configures-it-to-be-in-models-default-schema"></a><span data-ttu-id="fa0a5-895">DbFunction.Schema 为 null 或者空字符串将其配置为位于模型的默认架构中</span><span class="sxs-lookup"><span data-stu-id="fa0a5-895">DbFunction.Schema being null or empty string configures it to be in model's default schema</span></span>

[<span data-ttu-id="fa0a5-896">跟踪问题 #12757</span><span class="sxs-lookup"><span data-stu-id="fa0a5-896">Tracking Issue #12757</span></span>](https://github.com/aspnet/EntityFrameworkCore/issues/12757)

<span data-ttu-id="fa0a5-897">**旧行为**</span><span class="sxs-lookup"><span data-stu-id="fa0a5-897">**Old behavior**</span></span>

<span data-ttu-id="fa0a5-900">**新行为**</span><span class="sxs-lookup"><span data-stu-id="fa0a5-900">**New behavior**</span></span>

<span data-ttu-id="fa0a5-904">**为什么**</span><span class="sxs-lookup"><span data-stu-id="fa0a5-904">**Why**</span></span>
